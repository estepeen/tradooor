# Public Assets

Place your static assets here:

- **logo.svg** - Logo for the navigation bar
- **favicon.png** - Favicon for the browser tab

## File Locations

- `public/logo.svg` - Will be accessible at `/logo.svg`
- `public/favicon.png` - Will be accessible at `/favicon.png`

After adding files, restart the Next.js dev server for changes to take effect.
